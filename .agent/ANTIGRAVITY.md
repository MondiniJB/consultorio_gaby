# Configuración de Orquestación Global

Este agente está configurado para trabajar en el proyecto del consultorio del Dr. Gabriel Pellegrini, bajo los estándares de Qvanta Studio.

## Directrices Globales
1. Priorizar la experiencia del usuario y la accesibilidad en todas las decisiones técnicas.
2. Seguir estrictamente las reglas de diseño y experiencia de usuario (ver `skills/ui-ux-rules.md`).
3. Mantener la coherencia comunicacional y el tono adecuado en todos los textos (ver `skills/copywriting.md`).
4. Todo el desarrollo debe mantenerse modular, limpio y escalable.
