# Reglas de UI/UX y Diseño

1. **Tailwind CSS Estricto**: Todo el estilizado debe realizarse exclusivamente utilizando clases utilitarias de Tailwind CSS. Evitar el uso de hojas de estilo personalizadas.
2. **Acciones a un Clic**: La arquitectura de la interfaz debe estar diseñada para que el usuario logre sus objetivos principales con la menor fricción posible (idealmente en un solo clic). Interfaces directas y libres de distracciones.
3. **Minimalismo y Claridad**: Priorizar la legibilidad de la información médica y de contacto por encima de adornos innecesarios.
